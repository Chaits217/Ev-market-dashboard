# India's EV Market: Adoption Trends & Growth Analysis (2018–2024)

An interactive Power BI dashboard analyzing the growth of India's 
electric vehicle market from 2018 to 2024.


## Key Insights
- EV registrations grew **9.68x** between 2018 and 2024
- **Delhi** leads all states in total registered EVs
- **Four-wheelers** are the fastest-growing vehicle category

## What's Inside
- Year-over-year EV registration trend
- State-wise adoption comparison (top 10 states)
- Vehicle category share over time (four-wheeler, two-wheeler, 
  goods, public service)
- Interactive year filter

## Tools Used
Power BI · Power Query · DAX


## How to View
Download the `.pbix` file and open it in Power BI Desktop to explore 
the filters and charts.
